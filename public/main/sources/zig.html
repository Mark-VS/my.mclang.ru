Zig

Официальный сайт: ziglang.org

Скачиваем архив zig-x86_64-linux-0.16.0-dev.2193+fc517bd01.tar.xz:

tar -xvJf zig-x86_64-linux-0.16.0-dev.2193+fc517bd01.tar.xz
cd zig-x86_64-linux-0.16.0-dev.2193+fc517bd01

Установка:

# Просто копируем папку в систему:
sudo cp -r zig-x86_64-linux-0.16.0-dev.2193+fc517bd01 /opt/zig
sudo ln -s /opt/zig/zig /usr/local/bin/zig

# Проверяем:
zig env
# Посмотреть версию:
zig version

Удаление

sudo rm -fr /opt/zig
sudo rm -f /usr/local/bin/zig
