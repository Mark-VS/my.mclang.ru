Вернуться назад

sl

Страничка на GitHub: github.com/mtoyoda/sl

Форк: github.com/eyJhb/sl

Скачиваем исходники:

git clone https://github.com/mtoyoda/sl.git
# Или:
git clone https://github.com/eyJhb/sl.git

# Переходим в папку:
cd sl

Сборка

Собираем бинарник:

make

У нас появился бинарник sl. Мы можем сразу же его запустить:

./sl

Установка

У нас в Makefile нет цели install. Проверить можно так:

make install --dry-run
  # make: *** Нет правила для сборки цели «install».  Останов.

# или посмотрим с помощью grep:
grep Makefile install

Поэтому просто скопируем бинарник:

sudo cp sl /usr/local/bin/

# Или:
sudo install -m 755 sl /usr/local/bin/sl

Теперь программа установлена в систему. Её можно запускать вот так:

sl

Удаление

Просто удаляем бинарник:

sudo rm -f /usr/local/bin/sl
