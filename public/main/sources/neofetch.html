neofetch

Страничка на GitHub: github.com/dylanaraps/neofetch

Скачиваем исходники:

git clone https://github.com/dylanaraps/neofetch.git
cd neofetch

Поскольку neofetch написан на bash, то внутри уже имеется рабочий shell-скрипт: neofetch. Его сразу можно запустить:

./neofetch

Makefile здесь нужен только для установки. Можно туда зайти и посмотреть куда будет ставиться neofetch.

Установка

sudo make install

В систему скопировалось два файлика. Сам shell-скрипт попал в /usr/bin/neofetch, а man-страничка – в /usr/share/man/man1/neofetch.1. Проверить можно так:

# Проверяем где сам скрипт:
which neofetch

# Проверяем где man-страничка:
man --where neofetch

Следует обратить внимание: установка произошла не в пользовательскую /usr/local/bin/, а в системную /usr/bin/ папку. man-странички также установлись в /usr/share/man/man1/.

Теперь можно пользоваться программой:

neofetch --color_blocks off --ascii_distro debian

Итого:

# Скачиваем shell-скрипт:
git clone https://github.com/dylanaraps/neofetch.git
cd neofetch

# Устанавливаем его в систему:
sudo make install

# Удаление из системы:
sudo make uninstall
