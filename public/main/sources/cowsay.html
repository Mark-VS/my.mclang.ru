cowsay

Официальный сайт: cowsay.diamonds

Страничка на GitHub: github.com/cowsay-org/cowsay

Скачиваем perl-скрипт:

git clone https://github.com/cowsay-org/cowsay.git
cd cowsay

Или скачиваем архив c GitHub или с оф. сайта:

tar -xzvf cowsay-3.8.4.tar.gz
cd cowsay-3.8.4

Проверим:

./bin/cowsay "Привет!"

# укажем животное:
./bin/cowsay -f kitty "Привет"
./bin/cowsay -f moofasa "Привет"
./bin/cowsay -f tux "Привет"

# Или можно явно указать файл с говорящим: 
./bin/cowsay -f ./share/cowsay/cows/default.cow "Hello"
./bin/cowsay -f ./share/cowsay/cows/milk.cow "Hello"

Установка

sudo make install

Удаление

sudo make uninstall
