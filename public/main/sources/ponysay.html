ponysay

Официальный сайт: erkin.party/ponysay

Страничка на GitHub: github.com/erkin/ponysay

ponysay is written in Python 3.

В README.md указаны все необходимые зависимости и процесс установки.

Проверка зависимостей:

./dependency-test.sh

Установка

Бинарники ставятся в /usr/bin. Их три штуки: ponysay, ponythink и ponysay-tool.

./setup.py --freedom=partial install
   
# Или
sudo python3 setup.py --freedom=partial install

Если хочется поставить программу в папку ~/.local/, то можно воспользоваться аргументом --private:

python3 setup.py --freedom=partial --private install

Удаление

Не уверен, что это всё, что установилось. Но это всё, что я нашёл. Поэтому лучше ставить в ~/.local/.

sudo rm -f /usr/bin/{ponysay,ponysay-tool,ponythink}
sudo rm -rf /usr/share/ponysay
sudo rm -rf /usr/share/licenses/ponysay
sudo rm -f /usr/share/man/man6/{ponysay,ponysay-tool,ponythink}.6.gz
sudo rm -f /usr/share/info/{ponysay,ponysay-tool,ponythink}.info.gz

/usr/lib/python3.14/site-packages/ – Здесь тоже вроде нет никаких файлов.

/lib/python3.14/site-packages/, /lib64/python3.14/site-packages/ – здесь тоже.
