pfetch

Страничка на GitHub: github.com/dylanaraps/pfetch.

В Arch есть только chaotic-aur.

Скачиваем shell-скрипт:

git clone https://github.com/dylanaraps/pfetch.git
cd pfetch

Или скачиваем исходники отсюда:

tar -xzvf pfetch-0.6.0.tar.gz
cd pfetch-0.6.0

Во втором случае у нас не будет файла Makefile.

Установка

sudo make install

# Если у нас нет Makefile, то:
install -Dm755 pfetch /usr/bin/pfetch

Бинарник скопировался в /usr/bin/pfetch.

Теперь система видит программу:

pfetch
    
# С указанием ascii-лого:
PF_ASCII="gentoo" pfetch

Удаление программы:

sudo make uninstall

# Или если нет Makefile, то:
sudo rm -f /usr/bin/pfetch
