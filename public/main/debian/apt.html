Apt

# Поиск пакета в репозитории по имени:
apt search package_name

# Только те пакеты, которые начинаются с "package_name":
apt search ^package_name

# Найти точное совпадение:
apt search ^package_name$

# Или:
apt-cache search package_name

# Если установлен aptitude, то:
aptitude search package_name

Проверить, установлен ли какой-то пакет в системе:

# Примерно то же, что и "pacman -Q":
dpkg -l | grep nano

# Аналог "pacman -Qi":
dpkg -s nano

# Найти пакет, установленный через apt:
# Можно так:
apt list --installed nano

# или:
apt list --installed | grep nano

Установить vim:

sudo apt update
sudo apt install vim
