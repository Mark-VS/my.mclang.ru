systemd

юнит-файлы:


/usr/lib/systemd/system/   # юнит-файлы, установленные пакетами (основные, системные).
/etc/systemd/system/       # юнит-файлы, созданные пользователем или изменённые вручную.

/etc/systemd/system/       # директория для активации

Например, вот эта команда:

sudo systemctl enable dhcpcd.service
   # Created symlink '/etc/systemd/system/multi-user.target.wants/dhcpcd.service' → '/usr/lib/systemd/system/dhcpcd.service'.

создала символическую ссылку на системный dhcpcd.service в /etc/systemd/system/multi-user.target.wants/.
