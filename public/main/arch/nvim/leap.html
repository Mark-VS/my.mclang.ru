Leap

Страничка на Codeberg: codeberg.org/andyg/leap.nvim

Ставим leap.nvim:

-- ~/.config/nvim/lua/plugins/leap.lua

return {
  {
    'https://codeberg.org/andyg/leap.nvim',
    keys = {
      { 's', '(leap)', mode = {'n','x','o'} },
      { 'S', '(leap-from-window)', mode = 'n' },
    },
  }
}

И подключаем его в init.lua:

-- ~/.config/nvim/init.lua

-- ...
require("lazy").setup({
   spec = {
      { import = "plugins.leap" },
      -- { ... },
   },
   install = { colorscheme = { "habamax" } }
})
-- ...

Управление

  • s → прыжок только по видимой части экрана
  • S → прыжок в другое окно (если открыто несколько)

Далее вводим две буквы для поиска и видим куда можно прыгнуть.
