nvim-visual-multi

Страничка на GitHub: github.com/mg979/vim-visual-multi

-- ~/.config/nvim/lua/plugins/visual-multi.lua

return {
   {
      'mg979/vim-visual-multi',
      branch = 'master'
   }
}

Подключаем плагин в init.lua:

-- ~/.config/nvim/init.lua


-- Setup lazy.nvim
require("lazy").setup({
   spec = {
      { import = "plugins.visual-multi" },
      -- ...  
   },
   install = { colorscheme = { "habamax" } }
})

Основные команды:

  • Ctrl-n – войти в режим мультикурсора
  • Ctrl-Down / Ctrl-Up – create cursor vertically
