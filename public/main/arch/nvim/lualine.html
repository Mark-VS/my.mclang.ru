Lualine

Страничка на GitHub: github.com/nvim-lualine/lualine.nvim

Устанавливаем Lualine:

-- ~/.config/nvim/lua/plugins/lualine.lua

return {
    'nvim-lualine/lualine.nvim',
    dependencies = { 'nvim-tree/nvim-web-devicons' },
    config = function()
        require'lualine'.setup {
            options = {
		 theme = 'nord',
	    }
	}
    end
}

Подключаем Lualine в init.lua:

-- ~/.config/nvim/init.lua

-- ...
-- Setup lazy.nvim
require("lazy").setup({
   spec = {
      -- { ... },
      { import = "plugins.lualine" },
   },
   install = { colorscheme = { "habamax" } }
})
-- ...

Темы лежат здесь: ~/.local/share/nvim/lazy/lualine.nvim/lua/lualine/themes/
