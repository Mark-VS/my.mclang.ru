lsusb
   # ...
   # Bus 001 Device 007: ID 08bb:2902 Texas Instruments PCM2902 Audio Codec
   # ...

Проверим, что ALSA видит карту:

aplay -l
   # ...
   # card 2: CODEC [USB Audio CODEC], device 0: USB Audio
   # ...

Это UMC22 (PCM2902)

Проверим вход (микрофон):

arecord -l
   # ...
   # card 2: CODEC [USB Audio CODEC], device 0: USB Audio
   # ...

UMC22 видна как устройство захвата. То есть микрофон физически доступен системе.

В PipeWire можно добавить усиление:

sudo pacman -S easyeffects

Установим плагины. Иначе Gain, Compressor и Limiter не будут работать.

sudo pacman -S lsp-plugins

sudo pacman -S calf

# calf limiter должен работать лучше, чем lsp

sudo pacman -S lv2
#

После установки нужно перезапустить EasyEffects.

Преобразовать стерео в моно:

Треки - Микшировать - Стерео в моно

превратить моно в стерео:

Скопировать одну монодорожку (Правка - Копировать)

Добавить вторую монодорожку (Треки - Добавить новый - Монотрек)

Вставить сюда скопированную дорожку

В одной дорожке Панорама: Pan → Left, на другой дорожке Панорама: Pan → Right

Выделяем обе дорожки и: Треки (Tracks) → Микшировать (Mix) → Свести и обработать (Mix and Render)

Настройки компрессора

  • Threshold: −20 dB
  • Ratio: 3:1
  • Attack: 5 ms
  • Release: 100 ms
  • Make-up gain: +6 dB

Limiter:

  • Ceiling: −1 dB
