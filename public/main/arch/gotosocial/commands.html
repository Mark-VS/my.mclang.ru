Команды

Вызвать справку

gotosocial --help

# Справка по "admin account":
gotosocial admin account --help

Создать пользователя admin:

sudo -u gotosocial gotosocial \
  --config-path /etc/config.yaml \
  admin account create \
  --username admin \
  --email admin@gts.test \
  --password 'СЛОЖНЫЙ_ПАРОЛЬ'
# Если пароль будет слишком простой, то ...
# ... GoToSocial его не примет

Пользователь создался

# Посмотреть список аккаунтов:
sudo -u gotosocial gotosocial --config-path /etc/gotosocial/config.yaml admin account list
  # ...
  # user account                    approved admin moderator suspended confirmed
  # mark 01JGAG6R0YD2VH4564PX9VF2ME yes      no    no        no        yes

В выводе видно, что аккаунт создан. Он подтверждён, но он не администратор.

Выдаём права администратора:

sudo -u gotosocial gotosocial \
  --config-path /etc/config.yaml \
  admin account promote \
  --username mark

Опять проверяем права:

sudo -u gotosocial gotosocial --config-path /etc/gotosocial/config.yaml admin account list
  # ...
  # user account                    approved admin moderator suspended confirmed
  # mark 01JGAG6R0YD2VH4564PX9VF2ME yes      yes   yes       no        yes

Теперь можно попробовать добавить запись с помощью веб-интерфейса типа pinafore.social, или phanpy.social/.

Либо можно поставить десктоп-клиент Whalebird:

yay -S whalebird-bin
