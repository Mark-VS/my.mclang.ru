Java

Для установки Java необходимо установить один из следующих пакетов:

  • jre-openjdk
  • jdk-openjdk

JRE (Java Runtime Environment). Это среда выполнения Java — минимальный набор, который нужен только для запуска Java-программ. Содержит виртуальную машину Java (JVM), стандартные библиотеки и необходимые файлы для запуска программ.

JDK (Java Development Kit). Это набор для разработки программ на Java. Содержит всё, что есть в JRE, компилятор (javac), инструменты разработчика (javadoc и др.) и дополнительные библиотеки.
