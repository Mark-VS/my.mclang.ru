mpv

Конфигурационный файл:

~/.config/mpv/mpv.conf

Горячие клавиши

  • 0 - увеличить звук
  • 9 - уменьшить звук
  • s - скриншот (по умолчанию сохраняется в рабочую директорию)
  • f - fullscreen
  • x - увеличить задержку показа субтитров
  • z - уменьшить задержку показа субтитров
  • v - включить/выключить показ субтитров
  • m - включить/отключить звук
  • > - переместить стоп-кадр на один шаг вперёд
  • < - переместить стоп-кадр на один шаг назад
  • j - переключаем субтитры по порядку
  • Shift + j - переключаем субтитры в обратном порядке
  • Shift+G - увеличить размер субтитров
  • Shift+F - уменьшить размер субтитров
  • Shift+- - видео отключается, а звук продолжает проигрываться
  • Shift+# - переключение звуковых дорожек
  • Shift+I - информация

Или можно добавить в конфигурационный файл один или оба из следующих параметров:

# Фиксированный базовый размер (в «масштабированных пикселях» при высоте окна 720)
sub-font-size=45   # по умолчанию 38

# Масштабирование всего шрифта (включая ASS); может ломать тонкую вёрстку ASS
sub-scale=1.20

Скриншоты

По умолчанию скриншоты сохраняются в домашнюю папку. Изменить дефолтный путь можно в конфигурационном файле. Добавим:

screenshot-directory=/home/username/Pictures/mpv

Подгрузить внешнюю звуковую дорожку:


mpv video-file.mkv --audio-file=external_track.ac3

# Несколько дорожек:
mpv video-file.mkv --audio-file=rus.ac3 --audio-file=eng.ac3

# Автопоиск звук. дорожек:
mpv video-file.mkf --audio-file-auto=fuzzy
# Если у нас видеофайл и звук. файлы - с похожими именами:
# - airplane.rus.ac3
# - airplane.commentary.mka
# - airplane.tralala.mka
# то mpv их подцепит

# При этом настройку audio-file-auto можно прописать
# в конфигурационный файл ~/.config/mpv/mpv.conf:
audio-file-auto=fuzzy
# Тогда при запуске mpv сам найдёт дорожки.
