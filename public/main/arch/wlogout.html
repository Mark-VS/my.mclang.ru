wlogout

Для работы wlogout необходимо дать разрешение пользователю выключать систему. Иначе при попытке выключить/перезагрузить компьютер мы будем получать сообщение типа: "Access denied". Обычно wlogout вызывает команду: systemctl poweroff, или systemctl reboot. Поэтому нам надо установить polkit-gnome или lxqt-policykit и добавить в конфиг:

exec /usr/lib/polkit-gnome/polkit-gnome-authentication-agent-1

# или:
exec /usr/bin/lxqt-policykit-agent

После перезагрузки системы (Win+Shift+C) всё заработает. Проверим, чтобы на фоне работал демон polkitd.

# Конфигурационные файлы:
~/.config/wlogout/style.css - описывает внешний вид (цвета, шрифты, размеры)
~/.config/wlogout/layout - описывает кнопки (power, reboot, logout и т. д.)

~/.config/wlogout/style.css:

window {
    background-color: rgba(30, 30, 30, 0.9);
}

button {
    background: #1e1e2e;
    border-radius: 15px;
    margin: 10px;
    padding: 20px;
    border: 2px solid #cba6f7;
    color: #ffffff;
    font-size: 20px;
}

button:focus {
    background-color: #89b4fa;
    color: #000000;
}

Тут следует обратить внимание на то, что в блоке button для задания фона используется свойство background, а не background-color, как в блоке window.

window {
