Горячие клавиши

  • Перемещение между вкладками: Tab, Ctrl+i
  • Выбрать сортировку: F6 → PERCENT_MEM → Enter (либо можно просто кликнуть курсором по нужному полю)
  • Сортировать по CPU: Shift+P
  • Сортировать по MEM: Shift+M
  • Сортировать по PID: Shift+N
  • Сортировать по TIME: Shift+T
