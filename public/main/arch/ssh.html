ssh

Расположение конфигурационный файлов:

  • /etc/ssh/ssh_config – системный конфиг
  • ~/.ssh/config – пользовательский конфиг

Зайти на удалённый сервер:

# По имени домена:
ssh mark@domain-name.ru

# по IP-адресу:
ssh mark@185.221.152.12

# Каждые 10 сек. отправлять пинг:
ssh -o ServerAliveInterval=60 mark@185.221.152.12
  # нужно, чтобы соединение не разрывалось при простое
  
# Можно даже так:
ssh -o ServerAliveInterval=10 -o ServerAliveCountMax=3 root@193.187.94.218
  • ServerAliveInterval=10 – клиент будет отправлять пинг каждые 10 секунд
  • ServerAliveCountMax=3 – это сколько раз подряд можно не получить ответ

А чтобы каждый раз не указывать эти настройки при запуске ssh, можно указать их в конфиге:

# ~/.ssh/config

Host *
    ServerAliveInterval 10
    ServerAliveCountMax 3

Запустить процесс, который будет отвязан от терминала:

nohup node index.js > output.log 2>&1 &

Перезагрузить ssh:

sudo systemctl restart sshd
