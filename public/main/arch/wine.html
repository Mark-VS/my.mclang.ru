Wine

Установка:

sudo pacman -S wine wine-mono wine-gecko winetricks
  • wine-mono – это свободная реализация Mono, предназначенная для запуска программ на .NET в Wine. Она эмулирует работу .NET Framework (старый), позволяя запускать программы, написанные на C#, которые зависят от .NET.
  • wine-gecko – HTML / встроенный браузер
  • winetricks – Установка зависимостей (менеджер зависимостей)
  • vcrun – Visual C++
  • dotnet – .NET Core/5/6 (новый) !!! Новый .NET Core ставить в Linux – не в Wine !!! (sudo pacman -S dotnet-sdk)
  • dxvk – DirectX → Vulkan

Создать префикс:

WINEPREFIX=~/wine/myapp winecfg

Это создаст папку ~/wine/myapp и откроет winecfg только для этой программы.

Запуск программы из этого префикса:

WINEPREFIX=~/wine/myapp wine program.exe

префикс

Для обычного случая:

winecfg

Для отдельного префикса:

WINEPREFIX=~/wine/myapp winecfg

Для 32-битного префикса:

WINEARCH=win32 WINEPREFIX=~/wine/oldapp winecfg
