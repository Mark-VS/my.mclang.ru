Копирование

У каждого процесса есть umask – маска, которая обрезает права по умолчанию.

Когда создаётся файл, система берёт базовые права:

  • для файлов: 666 (rw-rw-rw-)
  • для директорий: 777 (rwxrwxrwx)

И затем применяет umask.

Проверить umask:

# На домашнем ПК (Arch):
umask
0022

# На сервере (Debian):
umask
0002
