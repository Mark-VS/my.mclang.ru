NodeJS

Страничка на Github: github.com/nvm-sh/nvm

Сайт (вроде бы ка неоффициальный): www.nvmnode.com

  • Если ставить Node через nvm, то npm и npx ставятся автоматически вместе с версией Node в папку ~/.nvim/versions/node/v20.20.2/bin/.
  • Если же устанавливать Node через pacman, то npm и npx не установятся. Они находятся в отдельном пакете. Чтобы были доступны npm и npx, нужно установить пакет npm. Они поставятся в /usr/bin/.

nvm

Устанавливаем nvm:

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.4/install.sh | bash

# Или:
wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.4/install.sh | bash

# Затем:
source ~/.bashrc

Установка NodeJS:

# Установить последнюю версию:
nvm install node

# Установить lts-версию:
nvm installl --lts

# Установить конкретную версию:
nvm install 14.7.0

# Посмотреть список установленных версий:
nvm ls

# Посмотреть список досутпных версий:
nvm ls-remote

# Проверяем:
node --version
  # v25.8.2

npm --version
  # 11.11.1

В этом примере видно, что я по умолчанию не использую nvm. У меня используется системный Node (25.9.0). Это понятно из этой строчки: ->     system (-> v25.9.0)

Как переключаться между версиями:

# Переключиться на версию 20.20.2:
nvm use 20

# Вернуться обратно на системную версию:
nvm use system


# Посмотреть куда установлена конкретная версия node:
nvm which 25.8

# Посмотреть, куда установлен системный Node:
which node
