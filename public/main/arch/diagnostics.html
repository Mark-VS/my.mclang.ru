iotop

iotop — это утилита для мониторинга ввода-вывода (I/O) на диске в реальном времени, похожая на top, но для операций с диском.

sudo pacman -S iotop
sudo iotop -oPa

  # -o или --only — показывать только процессы, которые совершают I/O-операции
  # -P — показывать только процессы (не потоки)
  # -a — показывать накопленную статистику (с момента запуска iotop)

диск

Проверить состояние. smartctl – часть пакета smartmontools.

# Проверьте SMART-статус
sudo smartctl -a /dev/sda
# Или для вашего конкретного диска
sudo smartctl -a /dev/disk/by-id/ata-HGST_HTS541010A9E680_JA1000102WT6VM

# Посмотреть все подключённые диски

sudo lshw -class disk   # находит только nvme
sudo lshw -class storage -class disk   # находит больше
