paru

Установка paru:


# Ставим инструменты:
sudo pacman -S --needed base-devel git

# Клонируем репозиторий paru:
git clone https://aur.archlinux.org/paru.git
cd paru

# Собираем и устанавливаем:
makepkg -si


# Либо можно сначала собрать:
makepkg -s
# makepkg скачает зависимости и соберёт .pkg.tar.zst в текущей папке

# А потом установить собранный пакет:
sudo pacman -U paru-2.1.0-1-x86_64.pkg.tar.zst
  • paru-2.1.0-1-x86_64.pkg.tar.zs - это основной готовый пакет, который устанавливается в систему. Его ставят командой: sudo pacman -U paru-2.1.0-1-x86_64.pkg.tar.zs
  • paru-debug-2.1.0-1-x86_64.pkg.tar.zst - это пакет с отладочными символами (debug info). Его обычно не нужно устанавливать, только если ты собираешься отлаживать paru с помощью gdb или подобных инструментов.
  • paru-2.1.0.tar.gz → это исходный архив, который AUR подгружает для сборки. Он хранится просто как копия исходников и в установке не участвует.
