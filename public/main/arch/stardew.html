Локальный мультиплеер

Для того, чтобы работал локальный мультиплеер, необходим плагин Stardew Multiplayer Server Mod. Его можно скачать с сайта nexusmods.com.

Но прежде, чем ставить мод, необходимо установить SMAPI. Это загрузчик(менеджер) модов. Официальный сайт: smapi.io. Скачиваем оттуда SMAPI в zip-архиве, распаковываем архив и запускаем скрипт install on Linux.sh. Указываем путь к установленной игре. В моём случае это дефолтный путь: ~/'GOG Games'/'Stardew Valley'/game/. Теперь, когда мы запускаем игру привычным образом, то она запускается через SMAPI. В процессе запуска игры SMAPI подгружает моды из папки ~/'GOG Games'/'Stardew Valley'/game/Mods/. У меня в этой папке уже есть ConsoleCommands и SaveBackup. Видимо, они поставились вместе с установкой SMAPI.

Теперь скачиваем Stardew Multiplayer Server Mod. Он представляет собой zip-архив. Распаковываем его. Внутри папка с тремя файлами: config.json, manifest.json и Stardew-Multiplayer-Server.dll. Закидываем эту папку в ~/'GOG Games'/'Stardew Valley'/game/Modes/. Теперь можно запускать Stardew Valley и мод подгрузится из папки ~/'GOG Games'/'Stardew Valley'/game/Mods/'Stardew Multiplayer Server Mod'/.

Если мод не работает, то можно изучить конфиг ~/.config/StardewValley/ErrorLogs/SMAPI-latest.txt.

# Распаковать SMAPI:
unzip 'SMAPI 4.3.2 4.3.2.zip' -d smapi 

# Распаковать Stardew Multiplayer Server Mod:
unzip 'Stardew Multiplayer Server Mod-20659-1-0-6-1733899146.zip' -d StardewMultiplayerMod
