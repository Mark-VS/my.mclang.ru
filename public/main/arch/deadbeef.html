Last FM Scrobbler

Edit -> Preferences -> Plugins -> last.fm scrobbler. Вводим логин и пароль и ставим галочку напротив "Enable scrobbler".

deadbeef-plugin-waveform-gtk3-git


yay -S deadbeef-plugin-waveform-gtk3-git

# Плагин появился в списке плагинов (Waveform Seekbar). Теперь мы можем добавить его весто стандартного Seekbar.
# Для этого надо перейти в режим настройки дизайна: View -> Design mode
# После этого кликаем правой кнопкой мыши по Seekerbar и ... ПКМ -> Replace with ... -> Waveform Seekbar

Если я случайно испортил интерфейс во время редактирования внешнего вида, то всё исправить можно в файле ~/.config/deadbeef/config.


# У меня блок, в котором содержался испорченный внешний вид, выглядел вот так:
gtkui.layout.1.9.0 {"type":"vbox","legacy_params":" expand=\"1 1 1\" fill=\"1 1 1\" homogeneous=1","children":[{"type":"playtb"},

# Просто его удаляем и перезагружаем DeaDBeeF. После перезагрузки внешний вид вернётся в прежнее дефолтное состояние.
